# nodejs-tsdns
Node.js TSDNS server с RestFul API

# Использование

Все запросы, выполняются через GET

**Список DNS зон**

Url: Домен:3000/list?api_key=...


**Добавить DNS зону**

Url: Домен:3000/add?api_key=...&zone=...&target=...


**Получить DNS зону**

Url: Домен:3000/get?api_key=...&zone=...

**Удалить DNS зону**

Url: Домен:3000/del?api_key=...&zone=...
